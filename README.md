# AHCNSY207 · Undertake Propagation Activities
## Conservation & Ecosystem Management Learning Resource (Tasmanian Context)

![Version](https://img.shields.io/badge/version-2.0-green.svg)
![License](https://img.shields.io/badge/license-CC%20BY--NC%204.0-lightgrey.svg)
![AHC](https://img.shields.io/badge/AHC-Training%20Package-blue)

An interactive, comprehensive web-based learning tool designed for Certificate III in Conservation and Ecosystem Management students. This resource contextualises the national unit AHCNSY207 (Undertake Propagation Activities) for Tasmanian native flora and the specific needs of the ecological restoration industry.

## Overview

This application transforms the traditional training package into an engaging, self-contained learning experience. It covers the entire propagation workflow, from ethical sourcing and planning to aftercare and record-keeping, with a strong focus on the unique requirements of Tasmanian species and conservation practices.

### Key Features

- **Contextualised Content:** Aligned with AHCNSY207 and the AHC31421 Certificate III in Conservation and Ecosystem Management, with a specific focus on Tasmanian biodiversity, endemism, and industry pathways.
- **Interactive Learning:**
    - **Knowledge Checks:** A 10-question quiz with instant feedback to test understanding of propagation methods.
    - **Workplace Scenarios:** Real-world problem-solving exercises covering disease outbreaks, seed dormancy, and biosecurity (e.g., Phytophthora management).
- **Comprehensive Database:** Searchable database of 65+ Tasmanian native plant species with detailed propagation notes, including pre-treatments, methods, and conservation status.
- **Practical Record Keeping:** Fully functional, browser-based propagation logs and environmental monitoring sheets with local storage for data persistence, export, and import capabilities.
- **Industry Best Practice:** Detailed guidance on hygiene protocols, ethical seed collection (20% rule), permit requirements, and career pathways.

## Getting Started

This is a static HTML/CSS/JS application. No server or installation is required.

1.  **Download:** Save the provided HTML file (e.g., `propagation-resource.html`) to your computer.
2.  **Open:** Double-click the file to open it in any modern web browser (Chrome, Firefox, Edge, Safari).
3.  **Use:** Navigate through the tabs to explore content, complete quizzes, and start logging your propagation activities.

All data entered into the records tables is saved locally in your browser and will persist even after you close the page.

## Using the Application

The interface is divided into the following key sections:

| Tab | Description |
| :--- | :--- |
| **0. About CEM Context** | Introduces the importance of propagation in Conservation and Ecosystem Management, links to the training package, and outlines the Tasmanian context and career pathways. |
| **1. Propagation Methods** | Details the five core methods (Seed, Cutting, Division, Grafting, Layering) with specific techniques, hormone usage, and media mixes tailored to Tasmanian natives. |
| **2. Planning & Ethics** | Covers parent plant selection, hygiene protocols, ethical collection guidelines, permits, and work plan components. |
| **3. Techniques & Tools** | Explores tools, rooting hormones, misting systems, and container selection, along with a tool maintenance guide. |
| **4. Aftercare & Monitoring** | Focuses on nutrition, pest and disease management, greenhouse environmental checks, and the hardening-off process. |
| **5. Interactive Quizzes** | Test your knowledge with a 10-question quiz on propagation concepts. |
| **6. Workplace Scenarios** | Apply your skills to realistic workplace challenges (e.g., disease outbreaks, seed germination failure). |
| **7. Species Database** | A searchable and filterable database of over 65 Tasmanian species with propagation specifics. |
| **8. Records & Logs** | Fully functional propagation and environmental logs. Data is saved locally. You can add, delete, filter, and export your records. |
| **9. Glossary & Resources** | A comprehensive glossary of propagation terms and links to key resources like NRE Tas, Understorey Network, and Florabank. |

## Intended Audience

- Students undertaking **AHC31421 Certificate III in Conservation and Ecosystem Management** or related qualifications.
- TAFE and VET trainers and assessors seeking a contextualised learning resource.
- Landcare, Bushcare, and community nursery volunteers.
- Entry-level staff in native plant nurseries and restoration companies.

## Technology

This resource is built using standard web technologies:
- **HTML5**
- **CSS3** (with Flexbox and Grid)
- **Vanilla JavaScript**
- **LocalStorage API** for data persistence

It is a fully self-contained, offline-capable application that does not require an internet connection to function after initial download.

## Disclaimer

This resource is intended as a learning aid and reference tool. While every effort has been made to ensure accuracy, users should always verify critical information against current industry standards, workplace procedures, and relevant legislation (e.g., Nature Conservation Act 2002) before undertaking practical work. Information regarding conservation status and permits should be confirmed with NRE Tas.
